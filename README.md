# React Chatbot 🤖

This is a simple chatbot project I built while learning React. The project was created by following a YouTube tutorial and is part of my journey to understand React fundamentals and gain hands-on experience with frontend development.

## 🚀 About The Project

This chatbot provides a basic chat interface where users can send messages and receive responses. The main goal of this project was to practice React concepts and learn how interactive web applications are built.

## 🛠️ Technologies Used

- React
- JavaScript
- CSS
- Vite

## 📚 What I Learned

While building this project, I learned:

- Creating and using React components
- Writing JSX
- Managing state in React
- Handling user input and events
- Conditional rendering
- Organizing a React project structure
- Building a simple interactive UI

## ⚙️ Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/react-chatbot.git
```

### Navigate to the project directory

```bash
cd react-chatbot
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```
